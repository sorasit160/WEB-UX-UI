"WEB-UX-UI" 
เป็น File Forntend อย่างเดียว

# สร้างโฟลเดอร์โปรเจกต์ด้วย Vite
npx -y create-vite@latest frontend --template react

# เข้าไปที่โฟลเดอร์ของโปรเจกต์
cd frontend

# ติดตั้ง Library พื้นฐานของ React
npm install

# ติดตั้งระบบเปลี่ยนหน้า (Router)
npm install react-router-dom

# ใช้คำสั่งนี้ทุกครั้งที่ต้องการรันหน้าเว็บ
npm run dev
